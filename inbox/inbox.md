## [2026-03-18] Проверить Google Drive Sync

**Проблема:** При синхронизации Google Drive в git попадают большие файлы (PDF 92MB, 127MB) которые блокируют push на GitHub.

**Задачи:**
- Разобраться что такое Google Drive Sync и откуда берутся эти файлы
- Проверить все противоречия и конфликты связанные с Google Drive sync
- Настроить .gitignore чтобы большие файлы не попадали в git

## [2026-03-18] Правило: большие файлы не пушить на GitHub

**Решение принято:** 
- Большие документы (PDF, DOCX, медиа) хранить на Google Drive или локально
- На GitHub пушить только карточки-описания документов (.md) со ссылками на оригиналы
- В .gitignore добавить правила для больших файлов
