# РП#12: Интеграция с Saby Presto API

## Цель

Подключиться к Saby Presto API для получения актуальных данных кофейни (цены, меню, остатки, продажи) и интеграции с RAG-ботом.

## Контекст

Saby Presto — система учёта кофейни. Содержит:
- Актуальные цены на все позиции
- Меню (блюда, напитки)
- Остатки ингредиентов
- Статистика продаж
- Данные о сотрудниках

Вчера нашли файл "Сравнение цен - Контур - Саби престо.csv" в Downloads/archive/ДЛЯ ШЕФА/

## Артефакт

1. **Документация по Saby Presto API**
   - Найти официальную документацию (saby.ru / СБИС API)
   - Изучить endpoints, авторизацию, формат данных

2. **Скрипт подключения к Saby API**
   - Python модуль для работы с API
   - Авторизация (API key / OAuth)
   - Базовые методы: получить меню, цены, остатки

3. **Интеграция в RAG-бот**
   - RAG-бот отвечает на вопросы о ценах из актуального источника
   - Автообновление данных в Pack при изменении цен

4. **Тестовый запрос**
   - Получить меню кофейни
   - Получить цены на топ-10 позиций
   - Проверить формат данных
