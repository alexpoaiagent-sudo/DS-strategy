# MGMT.ENTITY.001 — Контур аналитики кухни и бара (Аналитический контур)

## Определение

**Контур аналитики кухни и бара** — управленческая сущность, обеспечивающая расчёт экономики кухни и бара в реальном времени на основе входящих данных из нескольких источников.

## Источники данных

| Источник | Данные |
|----------|--------|
| Накладные | Поступление товаров, цены поставщиков |
| Продажи (POS) | Выручка по позициям, объём продаж |
| Калькуляционные карточки | Нормативная себестоимость блюд/напитков |
| Прайсы поставщиков | Актуальные закупочные цены |
| Зарплатные данные | Фонд оплаты труда кухни/бара |

## Флаги контроля

| Флаг | Условие срабатывания |
|------|---------------------|
| Рост цен поставщика | Цена входящей накладной > норматив на X% |
| Расхождение калькуляции | Фактическая себестоимость ≠ карточка |
| Хозрасходы | Списания сверх норматива |
| Остатки | Уровень остатков ниже/выше порогового значения |

## Роли

| Роль | Ответственность |
|------|----------------|
| Аналитик кухни | Контроль экономики кухни, расширение аналитики через отдельный контур |
| Менеджер смены | Интерпретация флагов, принятие оперативных решений |

## Связи

- **Источники данных:** PACK-kitchen (калькуляционные карточки, заготовки), PACK-bar (рецептуры напитков, прайс-лист)
- **Потребители:** PACK-management (принятие управленческих решений), DS-strategy (стратегическое планирование)

## Примечания

Текущий статус: концепт. Реализация требует интеграции с системой POS и автоматизации разбора накладных. См. РП-12 (интеграция Saby Presto) и РП по автоматизации накладных из Telegram.
